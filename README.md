Plant Disease Detector 🌿🔍

An AI-powered web application that detects plant diseases from leaf images using Machine Learning and Computer Vision.

Features

- Upload a plant leaf image.
- Identify plant diseases automatically.
- Display disease information and possible treatments.
- User-friendly interface for farmers, students, and researchers.

Technologies Used

- Python
- Machine Learning / Deep Learning
- OpenCV
- TensorFlow / Keras
- Goal

To help farmers and plant enthusiasts identify diseases early and improve crop health through technology.
